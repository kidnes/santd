# 更新日志

ant-design-san 严格遵循 Semantic Versioning 2.0.0 语义化版本规范。

# 发布周期

  • 修订版本号：每周末会进行日常 bugfix 更新。（如果有紧急的 bugfix，则任何时候都可发布）
  • 次版本号：带有新特性的向下兼容的版本。
  • 主版本号：含有破坏性更新和新特性，不在发布周期内。
